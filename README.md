# 🛡️ نظام مفتشي حماية المستهلك - دبي
### تطبيق ذكي لدعم العمل الميداني لمفتشي دائرة الاقتصاد والسياحة

هذا التطبيق مصمم لتسهيل مهام المفتشين في الميدان عبر توفير وصول سريع لمصفوفة القرارات القانونية وحاسبة الغرامات الفورية بناءً على التشريعات المحلية في إمارة دبي.

## 🚀 المميزات الرئيسية
* **⚖️ القرار الذكي:** مصفوفة تفاعلية لاختيار القطاع والحالة وإصدار التوجيه القانوني فوراً.
* **💰 حاسبة الغرامات:** أداة لتقدير قيمة المخالفات حسب نوع التجاوز المرصود.
* **📚 السند القانوني:** استعراض سريع للمواد القانونية المرتبطة بكل مخالفة (مثل المادة 7 والمادة 13).
* **📱 واجهة متجاوبة:** تعمل بكفاءة عالية على أجهزة الكمبيوتر والهواتف الذكية.

## 🛠️ التكنولوجيا المستخدمة
* **Python**: لغة البرمجة الأساسية.
* **Streamlit**: لإطار العمل وتصميم الواجهة.
* **Pandas**: لمعالجة بيانات مصفوفة المخالفات.

## ⚙️ طريقة التشغيل
لتشغيل المشروع محلياً:
1. قم بتحميل المستودع (Clone).
2. ثبت المكتبات اللازمة: `pip install -r requirements.txt`.
3. قم بتشغيل التطبيق: `streamlit run App.py`.
